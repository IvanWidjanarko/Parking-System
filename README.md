# Parking-System-New-Generation

(Project By : Ivan Widjanarko & Jonathan Elloy) Untuk memenuhi tugas mini project mata kuliah Pemrograman Lanjut â€“ 01 pada semester ini, maka kelompok kami membuat sebuah program Parking System. Latar belakang pemilihan program ini adalah karena di dalam aplikasi parkir cukup mudah untuk menerapkan Bahasa C. Aplikasi ini berfungsi untuk mendata kendaraan yang masuk dan keluar, memberikan nomor tiket, menghitung biaya parkir, serta memberitahukan jumlah lahan parkir yang tersedia. Kami berusaha untuk menerapkan materi pembelajaran bahasa C yang sudah diajarkan di kelas untuk membuat program ini.
