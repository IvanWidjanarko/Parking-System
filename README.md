# Parking System (Mini Project Proglan 01 (DTE FTUI))

Proyek ini disusun oleh Kelompok 6 Proglan 01
  1. Ivan Widjanarko (1806149806)
  2. Jonathan Elloy S. (1806148712)

Untuk memenuhi tugas mini project mata kuliah Pemrograman Lanjut â€“ 01 pada semester ini, maka kelompok kami membuat sebuah program Parking System. Latar belakang pemilihan program ini adalah karena di dalam aplikasi parkir cukup mudah untuk menerapkan Bahasa C. Aplikasi ini berfungsi untuk mendata kendaraan yang masuk dan keluar, memberikan nomor tiket, menghitung biaya parkir, serta memberitahukan jumlah lahan parkir yang tersedia. Kami berusaha untuk menerapkan materi pembelajaran bahasa C yang sudah diajarkan di kelas untuk membuat program ini.
